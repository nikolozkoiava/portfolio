# Nikoloz Koiava — Portfolio

Personal portfolio website built with Next.js, featuring a cosmic terminal-themed design with interactive animations.

## ⚡ Live

Coming soon — `nikolozkoiava.ge`

## 🛠 Tech Stack

- **Framework:** Next.js 15
- **Styling:** Tailwind CSS + Custom CSS
- **Animations:** Framer Motion + Canvas API + Three.js
- **Fonts:** JetBrains Mono + Space Grotesk

## ✨ Features

- 🌌 Interactive cosmic background with constellation lines
- ⭐ Stars that scatter away from your cursor
- 🔥 Cursor trail with click spark effects
- 💻 Working terminal with easter egg commands
- ⌨️ Command palette (`Ctrl+K`)
- 📱 Fully responsive with mobile burger menu
- 🎯 Active section tracking in navbar
- 🔝 Scroll-to-top button
- 📸 Portrait with scan-line hover effect
- ✨ Contact section with ambient glow animation

## 🎮 Terminal Easter Eggs

Type these in the hero terminal:

| Command | Effect |
|---------|--------|
| `help` | List all commands |
| `sudo hire me` | Fullscreen "Access Granted" animation |
| `matrix` | Amber matrix rain effect |
| `coffee` | ASCII coffee art |
| `hack` | Fake hacking progress bar |
| `neofetch` | System info display |
| `whoami` | Linux-style user info |
| `uptime` | Days since automating |

## 🚀 Run Locally

```bash
git clone https://github.com/nikolozkoiava/portfolio.git
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
